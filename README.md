# playwright-sample

## init

```
npm init playwright@latest
```

## CodeGen

```
npx playwright codegen demo.playwright.dev/todomvc -o todo-action.spec.ts
```

## Run

- all

```
npx playwright test
```

- target

```
npx playwright test todo-action.spec.ts
```

## Debug

```
npx playwright test todo-action.spec.ts --debug
```
